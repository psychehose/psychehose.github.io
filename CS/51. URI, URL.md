
## URI (Uniform Resource Identifier)

* URI은 인터넷상의 Resource를 나타내는 고유한 식별자 (문자열)
- 목적: Resource를 식별
* URL, URN을 모두 포함하는 상위 개념

## URL (Uniform Resource Locator)

* URL은 Resource의 위치를 명시하는 URI의 한 종류
* Resource를 찾는 방법으로 URI를 표시함 (Protocol, Host, Port, Path)

## URN (Uniform Resource Name)

 * URN은 Resource의 고유한 이름
 * Resource의 위치가 달라져도 이름은 고유함 (영구적 식별자)
 * 잘 사용 안함 - 고유한 ID지만 웹 환경에서 찾을 수 없음. 결국 URL로 변경해야하는데 이를 URN 해소라고 한다.
 * 예시) 책 ISBN
